# SuperStore Sales Analysis & Forecast Dashboard

This project presents an interactive business intelligence dashboard built using Power BI to analyze sales performance, profit trends, and future sales forecasting using the SuperStore dataset.

The dashboard helps in understanding sales patterns, customer segments, product categories, and regional performance.

---

## Project Overview

The objective of this project is to perform sales analysis and build an interactive dashboard that provides useful business insights.

Main goals of this project:

- Analyze overall sales and profit
- Identify top-performing product categories
- Understand regional sales distribution
- Evaluate shipping modes
- Forecast upcoming sales trends

---

## Tools & Technologies Used

- Power BI
- Data Visualization
- Data Analysis
- Forecasting
- Excel Dataset

## Sales Dashboard

This dashboard provides a complete overview of the business performance.

Key Metrics Displayed:

- Sales by Subcategory

- Monthly Sales Trends
- Profit Trends

## Key Insights

Some key insights derived from the dashboard:

- Consumer segment contributes the highest sales share.
- Technology category generates higher revenue.
- Standard shipping mode is the most frequently used.
- Sales show seasonal patterns across different months.
- States like Texas and Washington generate significant sales.
